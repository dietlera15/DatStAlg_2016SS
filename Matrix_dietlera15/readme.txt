MATRIX - Datenstruktur

Aufgabenbeschreibung:
---------------------
Implementieren Sie einen abstrakten Datentypen (Klasse Matrix), um Standardoperationen auf ganzzahligen Matrizen anwenden zu können.

Beschreibung der Klasse Matrix
------------------------------
Attribute:
- private int[][] daten; ... 2-dimensionaler int-Array zur Speicherung der Matrizenkoeffizienten
- private int numRows, numCols; ... Anzahl der Zeilen und Spalten 

Operationen:
-Matrizenaddition: add(Matrix)
-skalare Multiplikation: mult(int)
-Matrizenmultiplikation: mult(Matrix)
